# AI-OS Architecture

## 系统架构（第一版）

```
+-----------+     +-------------+     +-------+     +-------+
| ESP32-S3  | --> | Voice       | --> | AI    | --> | TTS   |
| (Mic + Spk)|    | Gateway     |     | Server|     | Engine|
|           |     | (FastAPI)   |     | (LLM) |     |       |
+-----------+     +-------------+     +-------+     +-------+
     ^                                                |
     |                                                |
     +------------------------------------------------+
                     Audio playback
```

## 模块职责

### ESP32 设备
- 麦克风录音 → 发送音频流
- 接收语音回复 → 播放
- WiFi 联网

### AI Server (FastAPI)
- 接收音频
- STT：语音转文字
- 调用 LLM 生成回答
- TTS：文字转语音
- 返回音频流

## 数据流

1. ESP32 录音 → 音频数据 → HTTP POST → Server
2. Server → STT → 文本 → LLM → 回答文本 → TTS → 音频
3. Server → HTTP Response → ESP32 播放

## 设计原则

- 模块化：每个组件可独立替换
- 本地优先：尽可能在本地运行
- 渐进式：从最简单的通路开始，逐步增强
