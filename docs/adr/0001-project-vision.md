# ADR-0001: AI-OS Project Vision

| 字段 | 值 |
|------|-----|
| 项目名称 | AI-OS |
| 决策 | 采用模块化架构设计 |
| 日期 | 2026-07-12 |
| 状态 | Accepted |

## 背景

目标是打造一个长期成长型 AI 数字伙伴，具备长期记忆、主动成长、
多终端接入、数字人格等能力。

## 决策

采用模块化架构设计。核心模块保持独立：

- **ESP32 设备层**：语音采集与播放
- **Voice Gateway**：音频流转换与路由
- **Brain**：LLM 交互与对话管理
- **Memory**：长期记忆存储
- **Agent**：主动能力

## 原因

- 未来需要支持 ESP32、手机、Web、Agent、Memory 等多种终端和能力
- 模块化使各组件可独立开发、迭代、替换
- 降低初始复杂度，从最小可用开始逐步演进

## 影响

- 初始开发成本略高（需定义模块接口）
- 长期维护成本大幅降低
- 支持渐进式增长，避免重写
